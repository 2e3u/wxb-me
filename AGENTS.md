# wxb.me Codex 行为准则

## 研习板块通用规则
1. 只改 `index.html` 内联内容与必要的本地资源；不得拆外部 JS/CSS，不得引入国外 CDN。
2. 任何已有 11408 深度内容都必须保留，新增内容只追加或精确锚点修改。
3. 含 LaTeX 的内容必须用 `R`` ` 的 String.raw 模板；普通内容也优先保持既有 `concept / visual / examples / pits / mn` 模型。
4. 改完必须本地预览验证：`window.STUDY` 可加载、对应考点可渲染、控制台无 error、移动端不明显溢出。
5. 部署前扫描外链：不得出现 `jsdelivr`、`googleapis`、`unpkg`、`cdnjs` 等 CDN。

## 四六级研习专项规则
1. 以中国教育考试网 CET4/CET6 笔试结构为骨架，确保写作、听力、阅读、翻译四块不漏。
2. 四级和六级分开建 subject；四级讲过级保底，六级讲难度升级与高压取舍。
3. 内容定位是“过级作战手册 + 官方题型知识库”，不是普通英语教材。
4. 每个考点都必须有实战例子、模拟题、可直接套用的步骤，以及必要基础知识点。
5. 每个题型都要讲合法应试抢分：不会完整做时怎么拿格式分、关键词分、定位分、表达分。
6. 写作要给高分句型、平替句型、救命句；必须解释适用边界，避免空泛万能模板。
7. 翻译要讲“不会翻译怎么办”：上位词替换、解释性翻译、拆长句、保主谓宾、舍弃难修饰。
8. 听力要讲“听不懂也能选”的信号策略：开头结尾、转折、重复、语气变化、选项同义改写。
9. 阅读要讲“定位偷分”：题干关键词、段首段尾、同义替换、绝对化选项、长难句主干。
10. 选词填空等低性价比题要明确时间上限和保底捡分法，不鼓励恋战。
11. 所有例文、例句、翻译段落、听力脚本必须原创，不搬课程、教材或真题原文。
12. “偷分”只指合法答题策略，不写作弊、泄题、违规押题或考试纪律规避。
