# wxb.me

Personal static website for `wxb.me`.

## Use On Windows

Clone the repository and open `index.html` in a browser.

```bash
git clone https://github.com/2e3u/wxb-me.git
cd wxb-me
```

The site is static. No build step is required for local preview.

## Files

- `index.html`: main page
- `rpg-character.html`: standalone character page
- `fonts/`: self-hosted fonts
- `mathjax-tex-svg.js`: local MathJax bundle

Vercel local project state and assistant worktrees are ignored.
